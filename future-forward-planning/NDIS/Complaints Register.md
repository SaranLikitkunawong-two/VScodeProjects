# Complaints Register
**SC Partnering Pty Ltd (trading as Future Forward Planning)**
*PM-POL-002 — Complaints and Feedback Management Policy*

> This register must be updated at each stage of the complaint handling process. Review quarterly for trends and systemic issues. Retain all records for a minimum of 7 years.

---

## How to Use This Register

- Assign a unique reference number to each complaint in the format: **CMP-[YEAR]-[NUMBER]** (e.g. CMP-2026-001)
- Open a new row when a complaint is received
- Update the row at each stage — do not create a new row for the same complaint
- Mark the complaint as **Closed** only when corrective actions are documented and the complainant has been notified of the outcome
- De-identify the complainant's name on request — use their reference number only

## Urgency Classifications

| Level | Description | Example |
|-------|-------------|---------|
| **Urgent** | Immediate risk to participant safety or wellbeing | Allegation of abuse or exploitation by a provider |
| **High** | Significant service failure or financial impact | Incorrect payment causing plan funds depletion |
| **Standard** | Service quality or communication concerns | Delayed response, unclear statements |
| **Feedback** | Suggestion or general comment, no grievance | Request for a different communication format |
