# Web Blog Design — Analysis

## Key Features
- Fixed navigation bar with logo, links, and call-to-action button
- Hero section with background image overlay, large headline, and action buttons
- Stats bar for quick facts (countries, stories, readers, years)
- Featured posts grid with cards (image, tag, title, summary, author, meta)
- Large featured post section
- Destinations grid with cards and overlays
- Newsletter signup section
- About author strip with avatar and bio
- Footer with brand, navigation, and legal links
- Responsive design for mobile and tablet

## Colour Scheme (CSS Variables)
- Cream: #faf7f2 (background)
- Sand: #e8ddd0 (borders, tags)
- Terracotta: #c0674a (primary accent, buttons)
- Terracotta Dark: #a0512e (button hover)
- Forest: #2e4a3e (headings, accents)
- Text: #2c2c2c (main text)
- Text Muted: #7a7a7a (secondary text)
- White: #ffffff (background, text)

## Key Objects & Design Types
- **Navigation:** Fixed, semi-transparent, blurred background, uppercase links
- **Hero:** Full viewport, gradient overlay, large serif headline, call-to-action buttons
- **Cards:** Rounded corners, subtle shadow, hover effects, image backgrounds
- **Sections:** Generous padding, clear section labels, serif headings
- **Buttons:** Pill-shaped, color transitions, primary/secondary styles
- **Footer:** Multi-column, dark background, white text, legal links
- **Responsive:** Grid and flex layouts adapt for mobile
